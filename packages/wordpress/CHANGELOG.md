# Changelog

All notable changes to the EveryCal WordPress plugin are documented in this file.

## [0.1.0] - 2026-04-09

### Added

- Initial public release of the EveryCal WordPress plugin.
- `EveryCal Feed` Gutenberg block (`everycal/feed`) with list, grid, and compact layouts.
- Plugin settings for default server URL, cache controls, event routes, creator profile links, and HTTP debug options.
- Event caching and prewarm behavior for feed and single-event rendering.
- Admin tools for HTTP log viewing and cache clearing.

### Notes

- This release establishes the first stable plugin package and documentation baseline.
